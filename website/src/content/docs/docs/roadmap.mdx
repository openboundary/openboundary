Planned features and standards support for upcoming releases.

## v0.2.0

| Feature | Status | Description |
|---------|--------|-------------|
| Claude Code Plugin | <span class="badge badge-soon">Next</span> | IDE integration that generates code within spec-defined boundaries, preventing middleware bypass and schema violations |
| Safezone System | <span class="badge badge-soon">Next</span> | Designated code regions preserved during regeneration. Mark custom handlers with annotations; compiler merges updates without overwriting |

## Contributing

Want to contribute? Check out the [GitHub repository](https://github.com/openboundary/openboundary) to get involved. See [releases](https://github.com/openboundary/openboundary/releases) for the latest version.

### Priority Areas

We're particularly interested in contributions for:

1. **New framework targets** - Adding support for Fastify, Chi, Axum
2. **Database providers** - Prisma integration, MySQL support
3. **Documentation** - Examples, tutorials, best practices
4. **Testing** - Edge cases, integration tests

### How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes with tests
4. Submit a pull request

See [CONTRIBUTING.md](https://github.com/openboundary/openboundary/blob/main/CONTRIBUTING.md) for detailed guidelines.
